## 📂 Tasks Included

### 🎯 Task 1: Number Guessing Game
- Generates a random number.
- User tries to guess the correct number.
- Provides hints (Too High / Too Low).
- Uses loops and conditional statements.

### 🧮 Task 2: Calculator
- Performs basic arithmetic operations.
- Supports addition, subtraction, multiplication, and division.
- Uses switch statements and user input handling.

### ❌⭕ Task 3: Tic Tac Toe Game
- Two-player console-based game.
- Displays a 3x3 board.
- Checks winning conditions.
- Uses arrays and logical conditions.
